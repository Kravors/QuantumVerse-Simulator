# Planet Textures Directory  
  
This directory contains NASA planet texture maps for celestial body rendering.  
  
## Available Textures:  
- sun.jpg: Solar surface  
- mercury.jpg: Mercury surface  
- venus.jpg: Venus surface  
- earth.jpg: Earth surface  
- mars.jpg: Mars surface  
- jupiter.jpg: Jupiter surface  
- saturn.jpg: Saturn surface  
- uranus.jpg: Uranus surface  
- neptune.jpg: Neptune surface  
- moon.jpg: Moon surface  
  
Note: Textures can be generated procedurally via ProceduralTextureGenerator if files are missing.
